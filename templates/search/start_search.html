Step 4: Find the match

It is an interactive search system.
In each iteration some shoes are shown. They are structured by columns.
Each iteration you are asked to understand if the columns differ by color or shape.
If they differ you should choose the most similar column to the seen shoe by color or shape respectively.
If you see the good match push the button "Buy something!" in the bottom of the page.
if you think that you will not find similar shoes push the button "I give up!" in the bottom of the page.
