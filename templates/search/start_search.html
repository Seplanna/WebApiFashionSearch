Step 4: Find the match

It is an interactive search system.
In each iteration some shoes are shown. They are structured by columns.
In each iteration, you are asked to understand if the columns differ by color or shape.
If they differ you should choose the most similar column to the shoe that you saw by color or shape respectively.
If you see the good match push the button "Buy something!" in the top of the page.
If you can't find something similar during 15 iterations you can push button "I give up!" in the top of the page.
The number of performed iterations is shown at the top of the page.
Some questions are spam detector. You should answer them correctly.
These questions look like normal ones, but there are no buttons "Buy something!" and "I give up!"
